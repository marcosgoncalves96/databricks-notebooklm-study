# Databricks Data Engineering Study Hub com NotebookLM

## Sobre o Projeto

Este repositório foi desenvolvido como parte do desafio **"Treinando uma IA de Aprendizagem: Explore o Poder do NotebookLM"** da DIO.

O objetivo foi utilizar o **NotebookLM** para aliar o pensamento crítico, curadoria de fontes e organização de conhecimento.

O Tema escolhido foi unir o desafio da DIO com uma situação real, utilizar a ferramenta para avançar nos estudos de algo que faz parte do meu dia a dia: **Databricks** , no qual será utilizado para estudar os principais conceitos da plataforma, utilizando documentação oficial como fonte de conhecimento.

Além de cumprir os requisitos do desafio, este projeto foi pensado para servir como uma base de estudos para Engenharia de Dados e até mesmo para preparação para a certificação **Databricks Certified Data Engineer Associate**.

---

# 🎯 Objetivos

- Explorar o NotebookLM como ferramenta de estudo.
- Aprender os principais conceitos do Databricks.
- Desenvolver habilidades em Engenharia de Prompts.
- Criar uma base de conhecimento reutilizável.
- Consolidar o aprendizado por meio de resumos e exercícios.

Durante este projeto foi possível compreender como o NotebookLM pode ser utilizado como um assistente de aprendizagem, conectando diferentes documentos oficiais e facilitando a compreensão de conceitos complexos.
